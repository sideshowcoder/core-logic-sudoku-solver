# Solve Sudoku using core.logic

## Sample puzzles

https://dingo.sbs.arizona.edu/~sandiway/sudoku/examples.html

## Run via leiningen

`lein run`

## Uberjar

```
lein uberjar
java -jar target/uberjar/solve-my-sudoku-0.1.0-SNAPSHOT-standalone.jar
```
